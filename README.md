This project is a dedicated, custom-themed website created to express love and apologies to your partner for Valentine's Week. It features a romantic pink aesthetic, interactive components, and celebratory animations.

## Features

- **Theme**: Full lovable pink and romantic colors with modern glassmorphism design.
- **Interactive Proposal**: A "Will you be my Valentine?" component.
- **Playful "No" Button**: The "No" button playfully evades the cursor, making it impossible to click.
- **Celebratory "Yes" Button**: Clicking "Yes" triggers a beautiful confetti animation and a loving message.
- **Apology Note**: A heartfelt "I am sorry" section.
- **Responsive Design**: Fully responsive and mobile-friendly.

## Tech Stack

- **React**: For building the user interface.
- **Vite**: For fast development and building.
- **Framer Motion**: For smooth animations and interactions.
- **Canvas Confetti**: For the celebration effect.
- **Netlify Ready**: Easy deployment configuration.

## How to Run Locally

1. Navigate to the project directory:
   ```bash
   cd valentine-app
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## Deployment on Netlify

1. Build the project:
   ```bash
   npm run build
   ```
2. The `dist` folder is ready for deployment.
